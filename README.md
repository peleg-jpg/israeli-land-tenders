# israeli-land-tenders

Israeli Land Authority (RMI) tender data, land allocation guidance, and bid process navigation. Use when user asks about Israeli land tenders, "michraz", "rashut mekarkei yisrael", RMI, Israel Land Authority, government land auctions, "haktzaah", land lottery ("hagralah"), "Dira BeHanacha" / "Mechir Lamishtaken", building rights, or state land allocation. Enhances remy-land-authority MCP server with tender process guidance and Hebrew terminology. Do NOT use for private real estate transactions (use israeli-real-estate skill instead) or non-Israeli land systems.

## Install

Will be installable via `npx peleg-skills install israeli-land-tenders` once the CLI ships.
Manual install: clone this repo into `~/.claude/skills/israeli-land-tenders/`.
